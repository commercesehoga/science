Mathematics Overview — 5 Days

CUET Maths — Key Facts
CUET Mathematics is one of the top 3 most-opted domain subjects. The paper contains 50 MCQs · 60 minutes · +5/–1 marking. All questions are compulsory. Paper is based on Class 12 NCERT Mathematics and tends to be moderate to difficult with a heavy focus on Calculus. Time management is critical.

Memory-Based Questions
Matrices · Q01
If A and B are symmetric matrices of the same order, then AB − BA is a:
ASymmetric matrix
BZero matrix
CSkew-symmetric matrix ✓
DIdentity matrix
Solution: For symmetric matrices A and B: (AB−BA)ᵀ = BᵀAᵀ − AᵀBᵀ = BA − AB = −(AB−BA). Since the transpose equals the negative, AB−BA is skew-symmetric.
Determinants · Q02
If A is a square matrix of order 3 and |A| = 5, find |3A|.
A15
B45
C135 ✓
D25
Solution: |kA| = kⁿ|A| for an n×n matrix. Here n=3, k=3: |3A| = 3³ × 5 = 27 × 5 = 135.
Integrals · Q03
Evaluate: ∫ 1/(1 + cos x) dx
Atan(x/2) + C
Btan(x/2) + C ✓
C−cot(x/2) + C
Dsec x + C
Solution: Use 1 + cos x = 2cos²(x/2). So ∫1/(2cos²(x/2))dx = (1/2)∫sec²(x/2)dx = tan(x/2) + C.
Application of Derivatives · Q04
The function f(x) = x³ − 3x is increasing on which interval?
A(−1, 1)
B(−∞, −1)
C(−∞,−1) ∪ (1,∞) ✓
D(0, ∞)
Solution: f'(x) = 3x²−3 = 3(x²−1). f'(x) > 0 when x² > 1, i.e. x < −1 or x > 1. Increasing on (−∞,−1) ∪ (1,∞).
Probability · Q05
A bag contains 3 red and 4 blue balls. Two balls are drawn at random. Find the probability both are red.
A1/7
B3/21 = 1/7 ✓
C6/49
D2/7
Solution: P = C(3,2)/C(7,2) = 3/21 = 1/7.

Continuity & Differentiability · Q01
If f(x) = x², find the derivative using first principles (limit definition).
Ax
B2x ✓
C2x + 1
Dx²
Solution: f'(x) = lim[h→0] [(x+h)²−x²]/h = lim[h→0] [2xh+h²]/h = lim[h→0] (2x+h) = 2x.
Integration · Q02
Evaluate: ∫ x·eˣ dx
Aeˣ + C
Bxeˣ + C
Ceˣ(x − 1) + C ✓
Deˣ(x + 1) + C
Solution: Integration by parts: ∫x·eˣdx = x·eˣ − ∫eˣdx = x·eˣ − eˣ = eˣ(x−1) + C.
Probability · Q03
Two cards are drawn at random from a deck of 52. What is the probability that both are kings?
A1/52
B1/221 ✓
C4/52
D1/169
Solution: P = C(4,2)/C(52,2) = 6/1326 = 1/221.
Vectors · Q04
If |a⃗| = 3 and |b⃗| = 4 and a⃗ · b⃗ = 0, find |a⃗ + b⃗|.
A7
B5 ✓
C1
D12
Solution: Since a⃗·b⃗ = 0 (perpendicular): |a⃗+b⃗|² = |a⃗|² + |b⃗|² = 9 + 16 = 25. So |a⃗+b⃗| = 5.
Linear Programming · Q05
In an LPP, the feasible region is bounded by the constraints x ≥ 0, y ≥ 0, x + y ≤ 4. The maximum of Z = 3x + 4y occurs at:
A(4, 0)
B(0, 4) ✓
C(2, 2)
D(0, 0)
Solution: Corner points: (0,0)→0, (4,0)→12, (0,4)→16. Maximum Z = 16 at (0,4).
3D Geometry · Q06
Find the distance between the parallel planes 2x + 3y − z = 4 and 4x + 6y − 2z = 10.
A1/√14 ✓
B2/√14
C3/√14
D√14
Solution: Rewrite: 2x+3y−z=5 and 2x+3y−z=4. Distance = |5−4|/√(4+9+1) = 1/√14.

Differential Equations · Q01
Find the general solution of dy/dx = y/x.
Ay = x + C
By = Cx ✓
Cy = x²/2 + C
Dy = Ce^x
Solution: Separable: dy/y = dx/x → ln|y| = ln|x| + ln|C| → y = Cx.
Inverse Trigonometry · Q02
Find the value of sin⁻¹(sin 5π/6).
A5π/6
Bπ/6 ✓
C−π/6
Dπ/3
Solution: 5π/6 is outside [−π/2, π/2]. sin(5π/6) = sin(π−5π/6) = sin(π/6). So sin⁻¹(sin 5π/6) = π/6.
Matrices · Q03
If A = [2 0; 1 3] and B = [1 2; 0 1], find AB.
A[2 4; 1 5] ✓
B[2 2; 1 3]
C[3 4; 1 5]
D[2 4; 0 3]
Solution: AB = [(2×1+0×0)(2×2+0×1); (1×1+3×0)(1×2+3×1)] = [2,4; 1,5]. Answer: [2 4; 1 5].
3D Geometry · Q04
The direction cosines of the line joining origin to (2, 3, 6) are:
A2/7, 3/7, 6/7 ✓
B2/6, 3/6, 6/6
C2/5, 3/5, 6/5
D1/7, 2/7, 3/7
Solution: Distance = √(4+9+36) = √49 = 7. DCs = (2/7, 3/7, 6/7).

Memory-Based Questions
Probability · Bayes' Theorem · Q01
A factory has machines A and B producing 60% and 40% of items respectively. Machine A produces 2% defective, B produces 3% defective. An item is defective — find P(it came from B).
A12/24 = 1/2 ✓
B12/27
C40/100
D3/5
Solution (Bayes): P(D|A)=0.02, P(D|B)=0.03. P(D) = 0.6×0.02 + 0.4×0.03 = 0.012+0.012=0.024. P(B|D) = 0.4×0.03/0.024 = 0.012/0.024 = 1/2.
Application of Integrals · Q02
Find the area enclosed between y = x² and y = x.
A1/6 sq. units ✓
B1/3 sq. units
C1/2 sq. units
D1/4 sq. units
Solution: Intersection: x=0,1. Area = ∫₀¹(x−x²)dx = [x²/2 − x³/3]₀¹ = 1/2 − 1/3 = 1/6.
Application of Derivatives · Q03
The rate of change of area of a circle with respect to radius when r = 5 cm is:
A5π
B10π ✓
C25π
D2π
Solution: A = πr². dA/dr = 2πr. At r=5: dA/dr = 10π.
Vectors · Q04
Find the cross product a⃗ × b⃗ if a⃗ = 2î + ĵ and b⃗ = î + 2ĵ.
A3k̂
B3k̂ ✓
C−3k̂
D0
Solution: a⃗×b⃗ = det[î,ĵ,k̂; 2,1,0; 1,2,0] = k̂(2×2−1×1) = k̂(4−1) = 3k̂.

Differential Equations · Q02
The order and degree of (d²y/dx²)³ + dy/dx = sin x are:
AOrder 1, Degree 3
BOrder 2, Degree 3 ✓
COrder 3, Degree 2
DOrder 2, Degree 1
Solution: Highest derivative = d²y/dx² → Order = 2. Power of highest order derivative = 3 → Degree = 3.
Determinants · Q03
If A is a matrix of order 3×3 and |A| = −4, find |adj A|.
A−4
B4
C16 ✓
D−16
Solution: |adj A| = |A|^(n−1) = (−4)^(3−1) = (−4)² = 16.
